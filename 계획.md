# 광고대행사 홈페이지 만들기

[광고대행사 홈페이지]

- 백엔드: 스프링부트2, java11
- 프론트 : vuejs
- 서버 : AWS EC2
- DB : AWS RDS(Maria DB)
- 페이지 구성
    - About(회사소개)
    - Work(광고포트폴리오)
    - Clients(고객)
    - Contact(연락)
    - recruit(채용)
- 주요기능
    - 게시판 기능
    - 로그인, 회원가입 기능
    - 파일 업로드 다운로드(영상,사진) 기능
    - 반응형 프론트
    - 카카오맵API를 통한 길찾기 기능
